# تطبيق المصحف الرقمي - أكاديمية أزهريون 📱

هذا المجلد يحتوي على ملفات تطبيق الأندرويد الخاص بالمصحف الرقمي، والمعد للبناء الفوري وتوليد ملف APK باستخدام **GitHub Actions** دون الحاجة لتثبيت أي برامج على جهازك.

---

## 🚀 طريقة البناء الفوري والحصول على ملف الـ APK:

1. **إنشاء مستودع جديد على GitHub:**
   - قم بالدخول إلى حسابك على GitHub وأنشئ مستودعاً جديداً (Repository) باسم من اختيارك، مثلاً: `quran-android-app`.

2. **رفع الملفات:**
   - ارفع **محتويات هذا المجلد (`quran-android-app`) بالكامل** (وليس المجلد الرئيسي نفسه) إلى المستودع الجديد على GitHub.
   - تأكد من رفع مجلد `.github` ومجلد `www` وملف `capacitor.config.json` و `package.json`.

3. **بدء البناء تلقائياً:**
   - بمجرد رفع الملفات على فرع `main` أو `master`، سيقوم GitHub Actions بالبدء في بناء التطبيق تلقائياً!
   - يمكنك متابعة حالة البناء من خلال تبويب **Actions** في مستودعك على GitHub.

4. **تنزيل التطبيق (APK):**
   - بعد انتهاء عملية البناء بنجاح (ستظهر علامة صح خضراء ✅)، اضغط على اسم التشغيل الأخير.
   - انزل لأسفل الصفحة إلى قسم **Artifacts** (المرفقات)، وستجد ملفاً مضغوطاً باسم `quran-app-debug`.
   - قم بتنزيله وفك الضغط لتجد ملف الـ APK جاهزاً للتثبيت الفوري على أي هاتف أندرويد!

---

## 🛠️ محتويات المجلد:
* **`www/index.html`**: واجهة تطبيق المصحف الرقمي كاملة وبدون أي أكواد PHP (تطبيق ثابت يعمل بالكامل محلياً).
* **`.github/workflows/build-apk.yml`**: ملف إعداد سير العمل الخاص بـ GitHub Actions والذي يقوم بتثبيت المتطلبات (Java, Android SDK, Capacitor) وبناء ملف الـ APK بشكل آلي وسحابي تماماً.
* **`capacitor.config.json`**: إعدادات تطبيق Capacitor (اسم التطبيق، معرف الحزمة com.azhareon.quran، وغيرها).
* **`package.json`**: ملف التبعيات المطلوب لبناء التطبيق.
* **`assets/icons/`**: أيقونات التطبيق الرسمية التي ستظهر على شاشة الهاتف.
